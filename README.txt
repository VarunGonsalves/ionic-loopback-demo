------------------------------------------------------------------------
This is a simple demo for connecting loopback server to an ionic 
application. The loopback server that connects to a mongoDB 
database. It communicates with the ionic app via REST API. The ionic 
application can be used to view, add, update or delete student data.
------------------------------------------------------------------------

PROJECT TITLE: Ionic-Loopback-Demo
PURPOSE OF PROJECT: To show a simple demonstration of connecting an 
		    ionic application to mongoDB via loopback api.
		    A student database is used.
VERSION or DATE: 1
HOW TO START THIS PROJECT: Start ionic-app by running ionic serve
			   Start loopback server by running npm start
AUTHORS: VarunGonsalves
USER INSTRUCTIONS: Have MongoDB running on 127.0.0.1:27107
                      ionic-app running on 127.0.0.1:8100
                         lb-app running on 127.0.0.1:3000
